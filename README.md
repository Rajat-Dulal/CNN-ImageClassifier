# CNN-ImageClassifier

## Implementation of CNN for Image Classification

Dataset used: Animal10

training_size = 0.8

validation_size = 0.2

Optimizer used: Adam (See "Results" and "Comparisons_optim" for comparision between multiple optimizers)

lr: 0.0001

patience_count = 3 (for early stopping)
